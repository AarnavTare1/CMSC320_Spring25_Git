

Q1: Why might we consider using Git? 

Answer: Git helps track changes in code, enabling collaboration among multiple
developers while maintaing a history of modifications. It also provides 
branching and merging capabilities, making it easier to expieriment
with new features without affecting the main codebase. 

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to various fields, including
	healthcare, finance, marketing, etc. For example, in healthcare
	data science can be used to predict disease outbreaks or 	
	personalize treatment plans. 


Q3: What is the difference between Merging and Rebasing?

Answer: Merging preserves branch history by creating a new commit that 
	combines changes, while rebasing rewrites history by moving 
	commits onto the target branch for a cleaner, linear structure.
